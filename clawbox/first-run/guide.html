🔌 聊天渠道配置指南

📱 Telegram

  1. 在 Telegram 中搜索 @BotFather
  2. 发送 /newbot 创建新 Bot，获取 Token
  3. 在 Dashboard → 设置 → Telegram 中填入 Token
  4. 点击保存，即可开始通过 Telegram 与 AI 对话

💡 提示：BotFather 会给你一个类似 123456:ABC-DEF1234 的 Token

💬 微信

  1. 微信个人号接入需使用第三方桥接工具（如 wechaty）
  2. 在 Dashboard → 设置 → 微信中配置桥接地址
  3. 企业微信用户可使用官方 API 接入

📄 飞书

  1. 在飞书开放平台创建应用
  2. 获取 App ID 和 App Secret
  3. 在 Dashboard → 设置 → 飞书中填入凭证
  4. 配置 Event 订阅 URL 为你的 Gateway 地址

🔧 API 直接调用

所有渠道都支持通过 REST API 直接调用：

curl -X POST http://127.0.0.1:18789/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"你好！"}'

❓ 常见问题

  • Dashboard 打不开？ 确保 Gateway 已启动（查看系统托盘图标），或重启电脑后重试
  • Node.js 冲突？ ClawBox 使用便携版 Node.js，不会影响系统已安装的 Node.js
  • 如何卸载？ 在开始菜单中运行 "卸载 ClawBox"，或从控制面板卸载
  • 如何修改 Agent 名字？ 在 Dashboard 设置中修改，或直接编辑 %APPDATA%\openclaw\openclaw.json
